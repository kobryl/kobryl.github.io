C++

Jeden z najpopularniejszych jezyków programowania. Wywodzi się bezpośrednio z języka C. Jest określany jako niskopoziomowy, co utrudnia pisanie aplikacji, ale znacznie zwiększa możliwości programistyczne.

Zastosowanie

C++ znalazł zastosowanie głównie przy programowaniu aplikacji desktopowych oraz sterowników.

Udogodnienia
  • Jezyk kompilowany - oznacza to, że pracuje bardzo szybko, nawet w aplikacjach z dużą ilością wierszy kodu.
  • Operacje na hardware - możliwość operowania na hardware jest możliwe dzięki wstawkom ASM.
  • Kompatybilność z C - jest kompatybily wstecznie z językiem C.
Dlaczego C++
  • aspekty niskiego poziomu
  • duża liczba bibliotek

Python

Obecnie najszybciej zyskujący popularoność język wysokiego poziomu. Jego ideą jest tworzenie czystego i przejrzystego kodu.

Zastosowanie

Python jest językiem ogólnego przeznaczenia. Możemy dzięki niemu tworzyć między innymi skrypty, aplikacje okienkowe, rozwiązania backednowe oraz sztuczną inteligencję. Używany jest głównie do rozwiązań niekomercyjnych.

Udogodnienia
  • Składnia - jest stworzona tak, by maksymalnie ułatwić jego czytelność, w kodzie nie występują średniki oraz klamry.
  • Dynamiczne typowanie - nie trzeba definiować typu zmiennej, gdyż Python sam je dobiera, można je zmieniać w czasie działania programu.
Dlaczego Python
  • Szerokie zastosowanie.
  • Szybko rozsnąca społeczność programistów - możliwość poradzenia się innych.

Java

Jest językiem programowania, stworzony przez firmę Sun Microsystems w 1995 roku. Obecnym właścicielem jest firma Oracle. Jest językiem szerokiego zastosowania.

Zastosowanie

Używany jest do tworzenia aplikacji desktopowych, aplikacji mobilnych oraz rozwiązań backendowych. Do działania wymaga zainstalowania bezpłatnego środowiska Java, który umożliwia uruchomienia programu w dowolnym systemie operacyjnym.

Udogodnienia
  • Garbage collector - jest to mechanizm odpowiedzialny za czyszczenie nieużywanych obiektów strefy, czyli pamięci na zarezerwowane obiekty, wirtualna maszyna sama decyduje kiedy wykonać czyszczenie.
  • WORA - write once, run anywhere - maszyna wirtualna Javy pozwala na uruchonienie programów na dowolnym systemie operacyjnym.
Dlaczego Java
  • Duża społeczność programistów, co ułatwia znalezienie rozwiązań problemów.
  • Duże zapotrzebowanie na rynku pracy na programistów Java.
  • Wieloplatformowość.
